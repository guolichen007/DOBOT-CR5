### 修改规则

- **禁止** 非等比缩放 PNG 纹理
- **禁止** 改变 Marker/Tag 真实物理尺寸
- **禁止** 修改 Marker/Tag ID
- 修改几何后必须运行 `validate_calibration_target_geometry.py`
- 修改纹理后必须运行 `validate_calibration_texture.sh`

## 内参与外参

### 内参来源

D455-like 内参定义在 `realsense_gazebo_description/config/d455_like_default.yaml`。
仿真中所有三台相机共用同一份内参。

### 外参初值估计

```bash
source /tmp/cr5_spray_simulation.env
rosrun cr5_spray_perception estimate_camera_extrinsics.py \
  --truth-source gazebo --output artifacts/calibration/extrinsics
```

- 单相机 PnP: ChArUco/AprilTag 检测 → solvePnP
- Gazebo truth: `/gazebo/model_states` 获取真实位姿
- 对比 PnP 估计值与真值，评估精度

### 可见性验证

```bash
rosrun cr5_spray_perception validate_calibration_visibility.py \
  --output artifacts/calibration/visibility
```

验证每台相机对每个面板的可见性。

## 待完成：多相机标定与重建

### 实施顺序

1. **三相机联合外参优化**: 多视角 PnP + 共同观测约束
2. **Bundle Adjustment**: 全局优化相机位姿和三维点
3. **同步 RGB-D 采集**: 三相机同一时刻采集 color+depth
4. **点云坐标转换**: 各相机点云 → 统一坐标系
5. **点云融合**: 多帧深度图拼接
6. **TSDF 三维重建**: 截断符号距离函数体素重建
7. **重建评价**: 与真值对比的定量指标
